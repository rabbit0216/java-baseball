## 📃 기능 목록
- [ ] 예외 사항 체크
  - [ ] 3자리수가 아닐 경우 애플리케이션 종료
    - [x] 자리수 체크
  - [ ] 1 ~ 9 가 아닌 값을 입력할 경우 애플리케이션 종료
      - [x] 자리별로 체크

- [ ] 컴퓨터 숫자 랜덤하게 저장
  - [ ] 1 ~ 9 사이의 숫자
  - [ ] 동일한 숫자 제외

- [ ] 플레이어로부터 입력받아 컴퓨터 숫자와 대조
  - [ ] 각 자리 숫자 대조
    - [ ] 숫자가 포함되어 있는지 확인
      - [ ] 있을 경우 위치 대조
        - [ ] 위치 동일 할 경우 '스트라이크' 출력
        - [ ] 다를 경우 '볼' 출력
      - [ ] 없을 경우 '낫싱' 출력
      
- [ ] 정답일 경우 게임 종료
  - [ ] 안내문 출력 후 종료
  - [ ] 플레이어로부터 재시작/종료 입력받아 처리
    - [ ] 재시작(1)일 경우 위의 과정 반복 
    - [ ] 종료(2)일 경우 애플리케이션 종료
